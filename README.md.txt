Getting and Cleaning Data Project Assignment

This repository contains the R code and the documentation files for the course " Getting and Cleaning the data" project provided the Coursera

Files:
1. CodeBook.md describes as how to use the variables, the data and any steps used to clean up the data
2. run_analysis.R contains the R code to preform the analyses in five steps. They can be launched in Rstudio just by importing the files.
3. secTidySet.txt is the output of the 5th step.
